Snake Battle – Human vs AI is a classic Snake game with a twist: your opponent is an AI that uses the A* search algorithm to chase food and flood-fill analysis to avoid getting trapped. Control your snake with the arrow keys and race to score 5 points before the AI does.
Features:

 AI uses A* pathfinding + flood-fill fallback for smart survival
 Real time two snake gameplay on a 30×30 grid
 You can speed up the AI snake as well
Visual AI path display (yellow dots)
 First to 5 points wins
 Instant restart without relaunching

Built with: Python · Pygame
